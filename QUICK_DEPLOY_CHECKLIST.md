## 🔧 **快速启动命令**

### 后端启动
```bash
cd backend
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

### 前端启动
```bash
cd frontend
npm run dev
```

### 数据库初始化
```bash
cd backend
alembic upgrade head
```

## 🚨 **常见问题快速修复**

### 1. 登录失败
- 检查数据库是否有管理员用户
- 确认密码是否正确(默认admin/admin123)
- 查看后端日志是否有错误

### 2. 页面空白或404
- 确认前端服务是否启动(localhost:3000)
- 检查后端API是否正常响应(localhost:8000/docs)
- 查看浏览器控制台是否有JavaScript错误

### 3. 数据导入失败
- 确认Excel文件格式是否正确
- 检查文件大小是否超出限制
- 查看后端日志获取详细错误信息

### 4. 工时计算错误
- 确认任务状态是否正确更新
- 检查工时计算规则配置
- 验证数据库中的工时记录

### 5. 导出功能异常
- 确认导出权限设置
- 检查文件生成路径权限
- 验证Excel文件是否正确生成

## 📞 **紧急联系方式**

如果遇到无法解决的问题：

1. **查看日志**:
   - 后端: `backend/logs/` 或控制台输出
   - 前端: 浏览器开发者工具控制台

2. **重启服务**:
   ```bash
   # 杀死现有进程
   pkill -f uvicorn
   pkill -f "npm run dev"

   # 重新启动
   ./deploy_ready_check.sh
   ```

3. **数据库重置** (紧急情况):
   ```bash
   cd backend
   alembic downgrade base
   alembic upgrade head
   python scripts/init_data.py  # 如果有初始化脚本
   ```
